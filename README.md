# 🔢 Real Image Digit Classifier using Machine Learning

A machine learning-based handwritten digit classification system built using **Python, Scikit-learn, TensorFlow, and PIL**.  
This project trains a **Random Forest Classifier** on the famous MNIST dataset and predicts digits from real uploaded images.

---

# 🚀 Features

- 📊 Trained on MNIST handwritten digit dataset
- 🤖 Machine Learning classification using Random Forest
- 🖼️ Real image digit prediction
- ⚡ Fast and lightweight implementation
- 📈 Accuracy around 96%
- 🧠 Beginner-friendly AI/Computer Vision project

---

# 🧠 Technologies Used

- Python
- NumPy
- Scikit-learn
- TensorFlow
- Pillow (PIL)

---

# 📂 Project Structure

⚙️ How It Works
Loads MNIST handwritten digit dataset
Trains a Random Forest model
Converts real images into grayscale 28x28 format
Predicts the digit from uploaded image

📸 Example Output
Accuracy: 96.50%
2.png: Digit 2

📊 Model Information
Model	Accuracy
Random Forest Classifier	~96%

🖼️ Image Requirements

For best prediction results:

Use clear handwritten digits
Prefer white digit on black background
PNG format recommended
Image automatically resized to 28x28
